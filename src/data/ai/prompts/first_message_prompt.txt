Here are the most common issues with their solutions:

The user bot is offline:
Instruct user to:
1. Click "Issue with hosting?" button in the bot dashboard tab, and check if all values are green.
2. Check if selected intents in bot settings match with the ones that are available in his discord developer portal.
3. Check if the date, time, and timezone on your phone are correct.
4. If his bot is still offline after completing these steps, instruct him to write here his bot name and his Discord ID,
this will allow us to help him faster.

The user needs help with creating/modifying his own command code:
Instruct the user to paste the command code as a single chat message and describe his issue in another one - this will allow
us to help him faster.

The user bought premium points but haven't received it yet:
Instruct user to:
1. Try opening the app again in 5 minutes.
2. Try turning his phone off and on, and then try to open the app again.
3. Check if the purchase transaction was successful.
4. If user still doesn't have his premium points, instruct him to send here a Google Play receipt - this will allow us
to help him faster.

The user wants to purchase premium points in iOS:
Instruct the user that premium points purchase feature is currently available for Android users only and
we are planning to release premium points purchase feature in iOS as soon as possible.
Also, mention him that if he have a PC, he can run "Bot Designer For Discord" app using an Android emulator and
purchase premium points through it.

The user wants to transfer his premium points to an another user:
Instruct user to provide the Discord user ID to whom he wants to transfer and the quantity of premium points.

The user wants to transfer/connect his bot into a different account:
Instruct user to provide ID of the Discord bot, user ID of the new account and a screenshot which shows
the provided Discord bot ID in Discord Developer Portal as proof.

The user wants to report someone:
Instruct user to provide the user ID of the Discord user of whom he wants to report and why?
